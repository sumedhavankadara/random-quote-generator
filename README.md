## Random Quote Generator with Adaptive UX

A minimalist web application that generates quotes based on category, mood, and daily psychological rhythm. Built using HTML, CSS, and JavaScript with responsive design, theme-of-day logic, animated transitions, and color psychology integration.

### Key Features
- Adaptive quote theming based on weekday
- Mood-based interaction (stressed / motivated)
- Smooth typewriter animation
- Dynamic background reflecting engagement tone
- Dark mode toggle
- Share quote functionality (Twitter/X)
